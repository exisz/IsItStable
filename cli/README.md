# 🔍 is-it-stable

Check if an npm package version is stable before you update.

[![npm](https://img.shields.io/npm/v/is-it-stable)](https://www.npmjs.com/package/is-it-stable)

## Install

```bash
npm install -g is-it-stable
```

## Usage

```bash
# Check the latest version of a package
is-it-stable next

# Check a specific version
is-it-stable react@19.0.0

# Find the latest stable version
is-it-stable next --stable
```

## How It Works

`is-it-stable` queries the [IsItStable.com](https://isitstable.com) API — a community-driven stability tracker for npm packages. Get real verdicts from developers who've already upgraded.
