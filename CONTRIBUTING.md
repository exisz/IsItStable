# Contributing to IsItStable

## Reporting Issues
Open a GitHub issue. Bug reports, feature requests, and package tracking requests welcome.

## Development
1. Fork the repo
2. `pnpm install`
3. `pnpm dev` — runs Next.js dev server
4. Make your changes
5. Open a PR against `main`

## Adding a Package
Want to track a new package? Open an issue with the package name and npm registry URL.

## Code Style
- TypeScript
- Tailwind CSS
- Prettier (default config)
